# 👷 Guide Chef de Chantier — RénoTrace Pro

**Rôle : Chef de chantier 👷**
Vous gérez un ou plusieurs chantiers, coordonnez votre équipe et suivez l'avancement.

---

## 🔐 Connexion
- Entrez votre identifiant et mot de passe
- Premier connexion → changez votre mot de passe immédiatement

---

## 🏗️ Gérer les chantiers

### Créer un chantier
1. Onglet **Chantiers** → bouton **＋** en haut à droite
2. Remplissez : nom, client, adresse, type de travaux, dates
3. Appuyez **Créer le chantier**

### Modifier un chantier
- Ouvrez le chantier → onglet **ℹ️ Infos** → bouton **✏️ Modifier**
- Mettez à jour le **pourcentage d'avancement** avec le curseur
- Changez le **statut** : En cours / En attente / Suspendu / Terminé

---

## 📸 Photos

- Bouton **📷** → sélectionnez le chantier, la surface, ajoutez une note
- Dans un chantier, onglet **📸 Photos** : toutes les photos groupées par jour
- Appuyez sur une photo → **agrandir** + **⬇️ télécharger**
- Vous pouvez **supprimer** une photo si besoin

---

## ⚠️ Signaler un problème

1. Dans le chantier → onglet **⚠️ Problèmes** → bouton **+ Signaler**
2. Titre, description, sévérité (Faible / Moyen / Élevé / Critique)
3. L'administrateur reçoit une **notification par email**
4. Suivez le statut : Ouvert / En cours / Résolu

---

## 📄 Exporter un rapport

- Dans un chantier → onglet **ℹ️ Infos** → **📄 Rapport**
- Génère un fichier HTML avec toutes les photos groupées par jour
- Imprimable ou partageable

---

## 📋 Tâches

- Sur l'**Accueil**, ajoutez vos tâches du jour
- Elles sont visibles par l'administrateur

---

## 🤖 Assistant IA

- Onglet **IA** : posez vos questions sur les chantiers
- Raccourcis : "Résumé des chantiers", "Qui a pris des photos aujourd'hui ?"
